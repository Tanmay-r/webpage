Projects

Look at my GitHub profile here.

8 Stage Pipeline Simulator For MIPS machine

  • Created a program in Java for simulation of the 8 stage instruction pipeline for MIPS machine. Implemented forwarding, stack and main memory and all the native instructions. Used Java Swing library for the GUI.
  • Completed as a course project, under the guidance of Prof. Bernard Menezes, IIT Bombay.

Election Statistics of India

  • Created a website to display election data of India's Lok Sabha elections. Used PosgreSQL and JDBC to maintain and access the database.
  • Completed as a course project for Database Systems course, under the guidance of Prof. Umesh Bellur, IIT Bombay.

Music Box

  • Created a model of music box placed in a room with a humanoid dancing figure which can be operated using keys. Its frames can be saved and animation can be created using it. It is implemented in C++ and uses OpenGL library to render graphics.
  • Completed as a course project for Computer Graphics, under the guidance of Prof. Parag Chaudhari, IIT Bombay.

MyPaint

  • Created an application to draw simple 2D drawings. It is implemented in C++ and uses OpenGL library to render graphics.
  • Completed as a course project for Computer Graphics, under the guidance of Prof. Parag Chaudhari, IIT Bombay.

Robustness Improvement of a Speaker Verification in Hearing Aids

  • Worked on the improvement of an existing prototype system for real-time speaker verification against robustness issues.
  • Based on an initial MATLAB framework, implemented an extension of the own voice speaker verification system driven by the signal-to-noise-ratio (SNR).
  • Optimized and evaluated the new speaker verification system for heading aids.
  • Completed as a summer research intern under the guidance of Prof. Tim Fingscheidt, TU Braunschweig, Germany during May-July, 2013.

Speech Recognition on Limited Vocabulary

  • Created a program in C++ that takes voice input and classifies it as ‘yes’, ‘no’, ‘save’ and ‘quit’.
  • Completed as a course project, under the guidance of Prof. Varsha Apte, IIT Bombay.

Automated Course Handler and Grader

  • Created a website which handles programming related lab courses. It provides an interface where students upload solutions to the assignments which are automatically checked and graded accordingly.
  • Used HTML and JavaScript for the front end and python libraries for the the back end.
  • Completed as a summer project in May-July 2012, under the guidance of Prof. Varsha Apte, IIT Bombay.
